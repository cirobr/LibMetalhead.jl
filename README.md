# LibMetalhead.jl

Adapted Metalhead.jl models.

### Networks:
* UResNet34((w,h), ch_in, ch_out), softmax (ch_out > 1) or sigmoid (ch_out == 1) output activation
* UResNet50((w,h), ch_in, ch_out), softmax (ch_out > 1) or sigmoid (ch_out == 1) output activation
